# FIDO Alliance WebAuthn demo

Slides: https://slides.com/fidoalliance/jan-2018-fido-seminar-webauthn-tutorial#/

### Requirements

- Firefox Nightly
- Nodejs + NPM
- Text editor

### Run

- `git clone https://github.com/fido-alliance/webauthn-demo/`
- `cd webauthn-demo`
- `npm install`
- `node app`

### Exposé SMB214

- Mise en oeuvre de la solution à l'aide des slides
- Publication sur une Web App Azure
